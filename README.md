# Plataforma-de-Streaming
Um sistema simples, apenas para estudo. 

## Como funciona
O sistema conta com duas telas: uma para cadastramento de filmes, e a outra com uma tabela onde os filmes são adicionados e também um sistema de busca por meio das categorias dos filmes.
O sistema vai ser criado usando JDBC, fazendo a aplicação comunicar-se com o banco de dados criado no MySQL, e assim ele realizar a inserção de dados por meio de uma interface com uso do JFrame Form.

## Etapas
O sistema foi feito em duas etapas:

### **1° Etapa:**
Criação das duas telas (cadastramento e tabela), na primeira, contando com um campo para identificação numérica, um campo para o nome do filme, um campo para a data de lançamento e a um campo para a categoria à qual ele pertence. Desenvolvimento do primeiro item de um CRUD, cadastrando os dados e fazendo conexão com o banco de dados.

### **2° Etapa:**
Aplicação das demais funcionalidades de um CRUD - desenvolver uma funcionalidade de consulta, uma para atualização e outra para exclusão dos dados - também sendo possível realizar um filtro dinâmico para a busca de categoria de filmes.
